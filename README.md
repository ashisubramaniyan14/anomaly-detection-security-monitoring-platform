# Anomaly Detection & Security Monitoring Platform

Cloud-based anomaly detection and system monitoring pipeline concept using AWS S3, Python data processing, ELK Stack dashboards, and automated alerting.

## What this repo will include
- Log ingestion + schema validation (batch)
- Anomaly detection (latency spikes / error bursts)
- ELK dashboards (Elasticsearch + Kibana)
- Optional AWS S3 storage integration

## Tech Stack
AWS S3 • Python • ELK Stack (Elasticsearch, Logstash, Kibana) • SQL

Initial setup complete. Implementation in progress.
